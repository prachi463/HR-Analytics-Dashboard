# HR-Analytics-Dashboard
1. 🚀 Overview
The **HR Analytics Dashboard** is an interactive data visualization project designed to analyze employee data and uncover insights related to workforce trends, attrition, and job satisfaction.  

This dashboard helps HR teams and decision-makers understand employee behavior, identify risk factors for attrition, and make data-driven strategic decisions.

---

2. 🎯 Purpose
The main goal of this project is to:
- Monitor employee attrition and retention trends  
- Analyze workforce demographics  
- Evaluate job satisfaction across roles  
- Enable data-driven HR decision-making  

---

3. 🛠️ Tech Stack
- **Power BI** – Data visualization & dashboard creation  
- **Microsoft Excel / CSV** – Data preprocessing  
- **DAX (Data Analysis Expressions)** – Calculations & measures  
- **Data Modeling** – Relationship building and transformation  

---

4. 📂 Data Source
- HR dataset containing employee details such as:
  - Age  
  - Gender  
  - Department  
  - Education Field  
  - Job Role  
  - Attrition Status  
  - Job Satisfaction Ratings  

> 📌 Note: This dataset is used for analytical and educational purposes.

---

5. 📊 Dashboard Highlights

### 🔹 Key Metrics

- **Attrition Rate by Gender across Age Groups**

---

6. ✨ Features
- 📌 Interactive filters (Education Level, Age Groups)
- 📌 Dynamic visuals for quick insights  
- 📌 Clean and modern UI design  
- 📌 Drill-down capability for deeper analysis  
- 📌 KPI cards for high-level overview  

---

7. 📈 Business Impact
This dashboard provides valuable insights that can help organizations:

- ✅ Identify high attrition departments  
- ✅ Improve employee retention strategies  
- ✅ Understand workforce demographics  
- ✅ Enhance employee satisfaction initiatives  
- ✅ Support strategic HR planning
